##Usage

###

Get first day of the month 

	DateTimeFormat::firstDayOfthisMonth();


Get last day of the month 

	DateTimeFormat::lastDayOfthisMonth());

Get First day of next month 		

	DateTimeFormat::firstDayOfNextMonth());
